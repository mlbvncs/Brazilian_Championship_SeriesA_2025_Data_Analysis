### Analysis
Descriptive analysis of the brazilian_championship_seriesa (2021-2025) dataset obtained through web scraping, seeking to identify if there are teams monopolizing Brazilian football based on the relationship between team + year and other variables.
### Recommended order for reading/consulting the folders:
1. notebooks/web_scraping.ipynb;
2. data/raw/brazilian_championship_seriesa.csv;
3. notebooks/analysis.ipynb;
4. powerbi.
